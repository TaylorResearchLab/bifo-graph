# KF-NBL Disease-Relevant Gene Reference Set (Observed-Variant Pool)
# §SM5.2 disease-relevant subset analysis — neuroblastoma germline predisposition
#
# Source: Kim J, Vaksman Z, Egolf LE, et al. (2023). Germline pathogenic
#   variants in 786 neuroblastoma patients. medRxiv 2023.01.23.23284864.
#   doi:10.1101/2023.01.23.23284864
#   PMCID: PMC9901064
#
# Specifically: 54 cancer predisposition genes harboring at least one
# pathogenic (P) or likely pathogenic (LP) germline variant in the 786
# neuroblastoma patients of the TARGET cohort, as reported in
# Supplementary eTable 3 of Kim et al. 2023.
#
# This is the conservative "observed-variant" disease-relevant pool — a
# strict subset of the 166-gene panel (kf_nbl_kim2023_panel_reference.txt).
# These 54 genes constitute empirical evidence: each has been observed
# carrying a pathogenic variant in an actual NBL patient. The remaining
# 112 panel genes had zero P-LP variants in 786 patients and are excluded.
#
# Pathogenicity was assigned by ClinVar evidence or modified InterVar
# assessment; variants were validated by Sanger sequencing where DNA was
# available.
#
# BARD1 is the most frequently affected gene (8/786 patients, 1.0%) and
# remains significant after Bonferroni correction in both control cohort
# comparisons (PMBB n=6,295 and gnomAD v2.1 non-cancer n=15,708).
#
# Format: one HGNC symbol per line. Comments (#) ignored.
# To be intersected with kf_nbl_seed_cuis.txt for §SM5.2 disease-relevant
# subset analysis (observed-variant pool variant).
#
# Co-author note: Sharon Diskin (Children's Hospital of Philadelphia) is
# senior author on the Kim 2023 paper and a co-PI on the U24 grant under
# which this BIFO paper is being conducted.
#
